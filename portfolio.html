The KG Store

The KG Store is a gym apparel e-commerce platform providing a seamless shopping experience for fitness enthusiasts worldwide.

Key Aspects and Technologies Used:

  • Front-End: React.js
  • Back-End: Node.js and Express.js
  • Database: MongoDB
  • Payment Integration: Stripe API
  • Deployment: AWS using Docker
ThoughtStream

ThoughtStream is a personal blog platform that enables users to create, edit, and share articles on various topics.

Key Aspects and Technologies Used:

  • Front-End: Angular
  • Back-End: Django REST Framework
  • Database: PostgreSQL
  • Authentication: OAuth 2.0
  • Responsive Design: Bootstrap
ChatConnect

ChatConnect is a real-time chat application that allows users to communicate with each other instantly.

Key Aspects and Technologies Used:

  • Front-End: Vue.js
  • Back-End: Node.js with Socket.IO
  • Database: Redis
  • Security: JWT for authentication
  • Deployment: Jenkins and Docker
